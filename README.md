First Memorama with login saved in the database with SwiftData

1.- It shows you a splash screen with animation.
2.- It asks for the user’s name to continue.
3.- Displays a Welcome view with a 5-second countdown.
4.- You can navigate to the game instructions.
5.- You can navigate to the Memorama game and have 180 seconds to find all the pairs.
5.1 Single grid of 12 positions.
5.2 You can listen to background music.
5.3 You can mute the background music with the sound button.
6.- It shows a congratulations screen when you find all the pairs.
7.- You can select Continue or Cancel on the congratulations screen.
7.1 Continue: Restart the game.
7.2 Cancel: Go back to the instructions screen.

Primer memorama con login guardado en base de datos swiftData.
1.- Se lanza un splash con animación.
2.- Se solicita un nombre de usuario para continuar.
3.- Se muestra una bienvenida y un conteo de 5 segundos para ingresar.
4.- Se navega a las instrucciones del juego.
5.- Se navega a el juego del memorama tienes 180 seg. para encontrar todas las parejas.
5.1- Unico grid de 12 posiciones.
5.2- Se escucha musica en background.
5.3- Puedes cancelar la musica de background con el boton de sonido.
6.- Se muestra una pantalla de felicitaciones al encontrar todas las parejas.
7.- Puedes seleccionar continuar o cancelar en la pantalla de felicitaciones.
7.1- Continuar: se reinicita el juego
7.2- Cancelar: se regresa a la vista de instrucciones.
